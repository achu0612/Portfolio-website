Recent Projects

ZIG-MET

[Winner out of 7K+ projects for the TNSI award]


A smart safety helmet for miners equipped with sensors for temperature, humidity, gas detection and an emergency button to save them from the toxic environmental conditions.

All helmets are connected among themselves and also to the base station in a mesh network using ZigBEE module.

SCANTEXT

Image to Text converter



[Tekhsol internship project]


An OCR based program that scans alphanumeric characters using image sensing and converts it to an editable text file.
Additional functionality of storing the data in a MySQL database is also provided.
